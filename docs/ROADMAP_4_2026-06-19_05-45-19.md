## Safety Boundary

Approving a request currently means:

```text
Sir approved the request in R.A.M.B.O. history.
```

It does not mean:

```text
R.A.M.B.O. connected an account, sent a message, deleted a file, spent money, or changed an external system.
```

That execution layer still needs to be built separately and guarded by Sentinel.
